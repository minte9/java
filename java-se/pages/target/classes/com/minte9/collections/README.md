## Collections

### Arrays

- core java.util
- fixed size
- store primitives (+ objects - wrappers unboxing)
- hard to manipulate

### Collections

- Java Collections Framework (JCF)
- Collection & Map (interfaces)
- List, Set, Queue (sub-interfaces)
- ArrayList, HashSet, ArrayDeque (implementations)

- dinamic size
- store objects  (+ primitives - wrappers allow primitives)
- rich functionalities (add, remove)
